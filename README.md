# FutureWorld Intelligence

A static GitHub Pages website for the `futureworld` repository.

## Brand Identity
FutureWorld Intelligence is an AI-powered cinematic educational media platform focused on climate, artificial intelligence, geopolitics, energy, water, food security and future systems.

## Deployment
Upload all files and folders to the root of the GitHub repository named `futureworld`, then enable GitHub Pages from `main` branch and `/root`.

Expected site URL:
`https://jamilahmad488.github.io/futureworld/`

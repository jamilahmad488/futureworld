# Practical AI & Website Building Course Traffic System

## Main Website Funnel

Primary course URL:

`https://futureworldintelligence.org/courses/practical-ai-website-building/`

Main discovery routes:

- Homepage mention or feature block
- Resources page priority card
- Future AI page learning section
- Individual lecture pages
- Social media posts
- YouTube or short video descriptions
- Substack or newsletter mentions

- Main landing page: `/courses/practical-ai-website-building/`
- Module pages: `/courses/practical-ai-website-building/modules/`
- Course resources: `/courses/practical-ai-website-building/resources/`
- Prompt library: `/courses/practical-ai-website-building/resources/prompt-library.html`
- Downloads and checklists: `/courses/practical-ai-website-building/resources/`

## Social Media Funnel

Use social media to create awareness, then direct learners to the website where the full course lives.

Funnel stages:

1. Awareness: Introduce the free beginner course and the problem it solves.
2. Education: Share small AI, GitHub, website and content workflow tips.
3. Module Promotion: Announce each new module when published.
4. Lecture Promotion: Share individual lecture releases with one clear benefit.
5. Conversion: Ask people to visit the course landing page and start learning.
6. Retention: Remind followers to continue with the next module or use a resource.

## Platform Roles

Facebook:

- Beginner-friendly course announcements
- Community learning posts
- Short explanations and direct website links

LinkedIn:

- Professional digital skills angle
- AI literacy, responsible publishing and career relevance
- Course launch and module milestone posts

X/Twitter:

- Short tips
- Threads explaining each module
- Quick course reminders with the course link

Instagram:

- Carousel posts for module summaries
- Reels for quick practical tips
- Story links and visual reminders

YouTube or Short Video Platforms:

- 30 to 60 second lesson teasers
- Screen walkthroughs
- Module overview videos
- Links back to the website course page

## Posting Schedule

Launch week:

- Day 1: Course announcement
- Day 2: Why beginners should learn AI and website publishing
- Day 3: Module roadmap post
- Day 4: Practical tip from Module 1
- Day 5: Reminder post with website link
- Day 6: Short video teaser
- Day 7: Recap and invitation to start

Ongoing schedule:

- 2 educational posts per week
- 1 course reminder per week
- 1 module or lecture announcement whenever new content is published
- 1 short video every 1 to 2 weeks

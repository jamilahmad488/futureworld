Module Purpose

Build a simple working environment

Who this module is for

Complete beginners who want to build a personal website, business website, NGO profile, blog, portfolio, course platform, research page or community project site.

What this module prepares

Your computer, accounts, software, folder system, learner prompts and basic understanding of local files, GitHub repositories and public website links.

Computer Requirements

Laptop or desktop specifications

Minimum

8 GB RAM, Intel Core i3 / Ryzen 3, 256 GB SSD and Windows 10 or 11.

Recommended

16 GB RAM, Intel Core i5 / Ryzen 5, 512 GB SSD and a 14-inch or larger screen.

Long-term

16-32 GB RAM, Intel Core i7 / Ryzen 7, 1 TB SSD and an external backup drive.

Learning Workflow

From beginner idea to public website

1. Prepare

Understand the course goal and prepare the right computer setup.

2. Install

Install the browser, VS Code, GitHub Desktop and Git for Windows.

3. Organize

Create clean folders for website files, images, notes, backups and social content.

4. Build

Create the first website pages with AI guidance and human review.

5. Publish

Move from local files to GitHub and then to a public website URL.

6. Share

Use the website as the central funnel for social media and future lessons.

Module 1 Lectures

Six beginner lessons

1.1

Mindset

Course Roadmap and Learner Mindset

Understand what you will build, how the course works and why beginners should not fear AI, websites, GitHub or digital tools.

Student Prompt

Act as a beginner-friendly digital skills teacher. I want to build a simple digital platform about [write your topic]. Explain what type of website I should create, what pages it should contain, and what tools I need in simple language.
1.2

Tools

Understanding the Tools Before Installing Them

Learn the role of ChatGPT, GitHub, GitHub Desktop, VS Code, browsers, Search Console, Analytics and domain registrars before using them.

Student Prompt

Explain these tools to me like I am a complete beginner: ChatGPT, GitHub, GitHub Desktop, VS Code, browser, Google Search Console, Google Analytics, and domain registrar. For each tool, tell me what it does, why I need it, and when I will use it.
1.3

Accounts

Creating the Essential Accounts

Create or prepare a Google account, ChatGPT account, GitHub account and later a domain registrar account. Use strong passwords and recovery options.

Student Prompt

Create a beginner checklist for setting up the accounts I need for building and publishing a website. Include Google account, ChatGPT account, GitHub account, and domain registrar account. Also include password safety and recovery tips.
1.4

Install

Installing the Required Software

Install only the essential beginner tools: Chrome or Edge, VS Code, GitHub Desktop, Git for Windows and optional screen recording software.

Student Prompt

Act as my beginner computer setup assistant. I am preparing my laptop for website building. Give me a simple installation checklist for Chrome or Edge, VS Code, GitHub Desktop, Git for Windows, and a screen recorder. Explain what each tool is used for.
1.5

Folders

Creating the Project Folder System

Create a clean folder system for website files, course notes, images, documents, downloads, backups, screenshots and social media content.

Recommended Structure

Digital_Projects - Practice_Website - Course_Notes - Images - Documents - Downloads - Backup - Screenshots - Social_Media_Content

Student Prompt

Help me create a clean folder structure for a beginner website project about [write your topic]. Include folders for website files, images, documents, downloads, backups, screenshots, and social media content. Also suggest simple file naming rules.
1.6

Links

Local Folder vs GitHub Repository vs Public Website

Understand the difference between a private laptop folder, an online GitHub repository and a public website URL. This prevents sharing local file paths with visitors.

Examples

Local: C:\Users\Student\Documents\Digital_Projects\Practice_Website Repository: github.com/username/project-name Website: https://username.github.io/project-name Domain: https://yourdomain.com

Student Prompt

Explain the difference between a local folder, a GitHub repository, and a public website URL as if I am a complete beginner. Give simple examples and explain why I should not share laptop file paths as website links.

Module Completion

Final outcome of Module 1

You should now have

  • A clear course roadmap
  • A selected project idea
  • Essential accounts created
  • Required software installed
  • A clean folder system

You should now understand

  • Local files are private
  • GitHub repositories store files online
  • Public websites need public URLs
  • AI assists, but the learner remains responsible
  • Simple prompts can remove confusion
